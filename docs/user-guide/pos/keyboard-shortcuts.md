# POS Keyboard Shortcuts

The POS terminal provides keyboard shortcuts for faster operations. You can trigger common actions without using the mouse.

## Priority and Customization

Keyboard shortcuts are fully customizable. The system resolves which shortcut to use based on a priority system:

1. **User Preferences (Highest Priority):** Each cashier can define their personal shortcuts for their own account.
2. **Register Defaults:** The manager can assign specific shortcuts to a physical register, overriding the system defaults.
3. **System Defaults (Lowest Priority):** The standard shortcuts provided out of the box.

## How to Customize

### Cashier-Level Customization
If you are logged into the POS Terminal, you can change your personal shortcuts:
1. Look for the **Keyboard Icon** in the top-right corner of the terminal screen.
2. Click it to open the **Keyboard Shortcuts** dialog.
3. Select an action you want to change, press the new key (e.g., `F4`), and save.
*Note: Your personal settings travel with your user account. If you log in to a different register, your shortcuts will still apply.*

### Register-Level Customization
If you are a manager and want to assign shortcuts that apply to anyone using a specific register:
1. Go to **Settings > Registers** (or **POS > Registers**).
2. Click **Edit** on the target register.
3. Scroll down to the **Keyboard Shortcuts** section.
4. Click **Configure** and map the keys you want to be the default for this register.

## Default Keyboard Shortcuts

If no overrides have been set, the following system defaults apply:

- **Pay (Tender):** `F12`
- **Void Line:** `Delete`
- **Apply Discount:** `F3`
- **Open Drawer:** `F8`
- **Hold Cart:** `F6`
- **Recall Cart:** `F7`
- **Search Items:** `F2`
- **Add Custom Item:** `F4`
