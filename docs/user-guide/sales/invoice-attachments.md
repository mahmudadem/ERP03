# Sales Invoice Attachments

Use invoice attachments to keep supporting files directly on the Sales Invoice record, so finance and operations teams can review evidence from one place.

Supported examples:
- Signed delivery proof
- Customer purchase order file
- Supporting spreadsheet or worksheet
- PDF backup document

## Limits and Rules

- Maximum **5 files** per invoice
- Maximum **10 MB** per file
- Allowed file types: **PDF, PNG, JPG, DOCX, XLSX**
- Attachments are stored per company (tenant-scoped) and cannot be shared across companies

## How To Attach a File

1. Open `Sales -> Invoices`.
2. Open the invoice you want to update.
3. In the **Attachments** card, click **Upload Attachment**.
4. Choose your file.
5. Wait for upload confirmation.

## How To Open or Remove a File

1. In the same **Attachments** card, find the file row.
2. Click **Open** to view/download the file.
3. Click **Remove** to delete the file from this invoice.

## Accounting and Audit Notes

- Attachments are evidence only and do not change posting values or journal entries.
- File metadata (who uploaded and when) is kept with the invoice record for traceability.
- Remove operations update the invoice attachment list and remove the file from storage.
