# Main Shell Sidebar

## What Changed

The production sidebar keeps the same navigation behavior, permissions, and menu structure, but accordion mode now has a cleaner Apex-inspired look.

## What Users See

- Accordion mode uses softer rows, clearer active highlights, and a thin nested rail for child links.
- Numbers, codes, and technical values in the main shell use JetBrains Mono where the screen marks them as mono/numeric text.
- Flyout/submenu mode still behaves as before.

## What Did Not Change

- Your visible modules and links still depend on your company, role, enabled modules, and workflow settings.
- Clicking a menu item opens the same page as before.
- No accounting, posting, tax, inventory, receivable, payable, report, or approval behavior changed.
- Apex is not being cut over as the production shell.
