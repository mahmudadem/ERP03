# Date Controls

ERP03 uses the same date control across operational pages and table filters. This keeps dates consistent with the company date format and gives users the same calendar behavior in Sales, Inventory, Accounting, and Purchases.

## Where this applies

- Stock Movements date filters.
- Stock Transfers transfer date.
- Sales Promotions valid-from and valid-to dates.
- Sales Price Lists valid-from and valid-to dates.
- Table date-range filters.

## How to use it

1. Click inside the date field.
2. Type a date using the company date format or choose a date from the calendar.
3. Use the calendar button to open or close the picker.
4. Right-click the field when available to use quick choices such as today, fiscal-year start, or period close.

## Notes

- This change does not alter stock posting, stock valuation, pricing rules, or promotion eligibility.
- Dates are still saved as ISO dates behind the scenes so reports and posting logic receive the same date values as before.
