# Master Data Lists Refresh After Save

When you add a new customer, vendor, inventory item, or warehouse from its master card, the list refreshes automatically after the record is saved.

This applies in both normal page mode and Windows mode. After saving, the card closes or returns to the list, and the new record should be visible without using the Refresh button manually.

If a search or type filter is active, the list still respects that filter. Clear the filter if the new record does not match the current view.
