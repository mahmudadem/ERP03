# Invoice Lists

ERP03 invoice list pages help users find and open existing invoices quickly.

## Sales Invoices

Open Sales -> Invoices.

You can:

- Create a new sales invoice.
- Filter by status.
- Filter by payment status.
- Filter by customer using the customer selector.
- Refresh the list.
- Clear active filters.
- Open an invoice from the row action.

Status and payment badges use consistent colors so users can scan invoices faster.

## Purchase Invoices

Open Purchases -> Invoices.

You can:

- Create a new purchase invoice.
- Filter by status.
- Filter by payment status.
- Filter by vendor using the vendor selector.
- Refresh the list.
- Clear active filters.
- Open an invoice from the row action.

## Financial Note

Invoice lists do not post or reverse accounting by themselves. They help users find invoices and open the detailed document page where posting, payment, cancellation, attachments, and audit actions are handled with the relevant controls.
