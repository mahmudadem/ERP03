# Accounting Voucher and Item Lists

## Accounting Vouchers

Open Accounting -> Vouchers.

You can:

- Filter vouchers by search text, type, form, status, and date range.
- Refresh the list.
- Create a new voucher if you have permission.
- Open a voucher from the table.
- Use available row actions such as approve, post, print, reverse, or delete based on the voucher status and your permissions.

Financial lifecycle actions remain controlled from the voucher table and voucher detail/modal behavior.

## Inventory Items

Open Inventory -> Items.

The same company item catalog is also available from Sales, Purchases, and POS
when the user has that module's item permission. In Purchases, open
Purchases -> Products & Services.

You can:

- Create a new item from the page header.
- Quick-add a simple item.
- Search by code or name.
- Filter by item type.
- Refresh or clear filters.
- Open an item from the row action.

Item status is shown as a badge so active and inactive items are easier to scan.

From Purchases, users can now list, search, open, create, edit, activate, and
deactivate items without receiving an “Endpoint not found” message.

## Financial Note

Item master records can affect downstream sales, purchase, inventory, costing, and posting behavior. Use Quick Add only for simple setup. Open the item card for full accounting, stock, and pricing configuration.
