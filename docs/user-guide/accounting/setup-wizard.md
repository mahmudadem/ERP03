# Accounting setup wizard

The Accounting setup wizard prepares a company for accounting use. It guides you through:

1. Fiscal-year start, end, and reporting periods.
2. The company base currency.
3. A chart-of-accounts template.
4. Default voucher types and form variants.
5. A final review before initialization.

The wizard follows the selected interface language in English, Arabic, or Turkish. Arabic uses
right-to-left layout.

Initialization can be completed only once. Later changes are made from Accounting Settings.
