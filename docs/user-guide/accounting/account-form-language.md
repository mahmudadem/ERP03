# Account form language

The Create Account and Edit Account window follows your selected application
language. Its tabs, fields, account classifications, accounting rules, currency
policy, approval controls, status messages, and action buttons are available in
Arabic, English, and Turkish.

Changing the interface language changes only the displayed wording and reading
direction. It does not change the account code, account classification, balance
nature, currency policy, posting behavior, or any existing accounting data.
