# Account Statements

Account statements show opening balance, dated ledger movements, and closing balance for
an account. Customer and vendor statements use the same accounting ledger so their balances
remain consistent with the general ledger.

If a statement previously displayed a system error, refresh the page after the production
index deployment and run the report again. No transaction recreation or reposting is needed;
the correction changes how existing ledger records are retrieved and does not alter balances.
