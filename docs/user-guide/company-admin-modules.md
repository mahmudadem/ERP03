# Company Admin Modules

## What This Page Does

The Modules page lets a Company Admin enable or disable modules that Super Admin has granted to the company.

## How Module Access Works

1. Super Admin grants a module to the company.
2. Company Admin opens Company Admin > Modules.
3. Company Admin enables the granted module.
4. The module appears in the tenant sidebar and its pages become available to allowed users.

## If No Modules Appear

If the page says no modules are available, the company may not have any granted modules for its plan.

If the page shows an error, the module list could not load. Refresh the page or ask a system administrator to check the company access and backend logs.

## Who Can Access This

Company owners and company administrators can manage company modules. Super Admin manages the grants from the platform side.
