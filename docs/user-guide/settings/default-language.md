# Default application language

ERP03 opens in Arabic for a user who has not selected a language before.
Arabic right-to-left layout is applied from the first screen.

If you manually select English, Arabic, or Turkish in your user preferences,
ERP03 remembers that choice on this browser and in your saved user preferences.
Your saved choice takes priority over the Arabic default.

This change affects only the interface language and layout direction. It does
not change company data, accounting entries, currencies, dates, or reports.
