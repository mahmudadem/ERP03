# System Settings

The System Settings page is the main place to find company configuration.

Settings are grouped by what they control:

- General: company details, appearance, menus, and notifications.
- Workflow: approval workflow plus Sales and Purchase document flow settings.
- Accounting and Tax: accounting defaults, currencies, inventory accounting, and tax codes.
- Access and Advanced: roles, user assignments, AI settings, and advanced administration.

## How To Use It

1. Open Settings from the sidebar.
2. Choose the group that matches the business area you want to configure.
3. Open the related settings page.
4. Make changes on the module settings page.
5. Use the save bar if the page shows unsaved changes.

## Who Can Use It

The hub can show links to configuration areas, but each destination page still applies its own permissions. If a user does not have permission for a setting, the protected page remains unavailable.

## Important Controls

Some settings affect financial behavior:

- Sales and Purchase workflow settings can change whether users work directly from invoices or through orders and receipts.
- Accounting settings control posting defaults and financial policies.
- Tax codes and currencies affect document and report calculations.
- Roles and user assignments affect who can approve, post, or administer records.

Change these settings carefully and test the affected workflow after saving.
