# Language and translation behavior

ERP03 supports English, Arabic, and Turkish interface text. Arabic sessions use RTL layout.

Production QA found untranslated English labels in two areas:

- **Sales Dashboard** — overview title, KPI cards, recent invoice/order tables, top-client panel, quick navigation, and recent activity.
- **Purchase Settings → Default Accounts** — AP sub-account generation labels, help text, backfill button, parent-account label, and account-code format presets.

These labels are now translated in Arabic and Turkish. If you still see English text in an Arabic or Turkish session, record the page and screenshot so it can be added to the remaining translation audit queue.
