# Module setup wizards

Accounting, Inventory, Sales, and Purchases now use the same setup layout.

## What you will see

1. A progress bar at the top shows the current setup step.
2. The middle area contains the settings for that step.
3. Back and Next remain in the same place throughout setup.
4. The final button initializes the selected module.

The layout follows the selected language direction. In Arabic, navigation and alignment are
displayed right-to-left.

## Before completing setup

Review account selections, workflow mode, numbering, and inventory behavior carefully. Completing
the wizard can create module defaults, but the visual layout does not change the accounting rules
or values you selected.

## Purchase setup

Opening an uninitialized Purchases module now goes directly to the setup wizard without showing the
expected uninitialized state as an error notification.
