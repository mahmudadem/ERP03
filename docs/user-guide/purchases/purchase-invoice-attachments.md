# Purchase Invoice Attachments

Purchase Invoice attachments let you keep the vendor's bill scan, delivery paperwork, or supporting spreadsheet linked to the exact bill in ERP03.

## What attachments do

- Store supporting evidence on a Purchase Invoice.
- Help reviewers and accountants verify the source document later.
- Keep the file tied to the same tenant/company as the invoice.

Attachments do not change accounting amounts, tax, AP balance, inventory valuation, payment status, or posting logic.

## How to use

1. Open `Purchases -> Invoices`.
2. Open a saved Purchase Invoice, or start a new Purchase Invoice.
3. Find the `Attachments` section.
4. Click `Upload Attachment`.
5. Select a PDF, PNG, JPG, DOCX, or XLSX file.
6. If the invoice is new and unsaved, the files are queued on the page and uploaded automatically when you save the invoice.
7. On a saved invoice, use `Open` to view a file.
8. Use `Remove` to delete a file from the invoice.

## Limits

- Maximum files per invoice: `5`
- Maximum file size: `10 MB`
- Allowed file types: PDF, PNG, JPG, DOCX, XLSX

## Who should use it

- Purchasing staff can upload the vendor bill or quote support.
- Accountants can check the source evidence before posting or paying.
- Managers can review documents without leaving the Purchase Invoice page.
