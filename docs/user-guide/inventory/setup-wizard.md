# Inventory setup wizard

The Inventory setup wizard prepares the company for stock operations. It guides you through:

1. Inventory accounting mode when Accounting is enabled.
2. The default warehouse.
3. Cost currency and optional Inventory Asset and COGS account mappings.
4. Negative-stock and automatic item-code policies.
5. A final review before initialization.

The wizard follows the selected interface language in English, Arabic, or Turkish. Arabic uses
right-to-left layout.

If Accounting is not enabled, Inventory can still track quantities and costs, but it does not
create financial or general-ledger postings.
