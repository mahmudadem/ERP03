# Inventory — Unsettled Costs report

Use **Inventory → Reports → Unsettled Costs** to review stock-out movements that are still waiting for a final cost basis.

The report can be filtered by date range, item, warehouse, and cost basis. The page shows:

- unsettled quantity,
- cost basis used or missing,
- total cost in base currency,
- the movement date, item, warehouse, and movement type.

Arabic and Turkish sessions show translated report titles, filters, summary totals, table columns, and empty-state messages.
